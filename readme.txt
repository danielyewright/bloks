Bloks is a full site editing block theme for everyone who wants to build creative, individual WordPress websites with ease.

== Description ==

Bloks is a Full Site Editing block theme for the WordPress block editor. In combination with the Bloks Blocks plugin you can easily create advanced page layouts using Bloks's pre-designed block patterns. Create a unique online shop, business website, portfolio or personal blog with Bloks and stand out from the crowd.

You can find more information and helpful resources about Bloks on https://bloks.io.
